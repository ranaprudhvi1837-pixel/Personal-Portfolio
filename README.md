# Rana Prudhvi Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, and contact information in a clean and professional format.

##  Features

- Responsive design (mobile + desktop friendly)
- Smooth navigation with sections:
  - Home
  - About Me
  - Skills
  - Projects
  - Contact Form
- Interactive UI with clean animations
- Social media links (GitHub, LinkedIn, Instagram, Twitter)
- Dark mode toggle (if enabled in JS)
- Project showcase section

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
